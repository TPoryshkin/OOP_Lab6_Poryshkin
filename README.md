# OOP_Lab6_Poryshkin
Лабораторна робота №6
Робота з колекціями об'єктів у C#
Мета роботи
Навчитися використовувати колекції об'єктів для зберігання, пошуку, видалення об'єктів.

Вимоги до виконання
Використання колекції List<T> для зберігання об'єктів

Реалізація операцій додавання/видалення/пошуку через методи List<T>

Розділення логіки програми та інтерфейсу користувача

Створення unit-тестів для перевірки методів

Забезпечення повного проходження всіх тестів

Детальне тестування програми

Опис проекту
Архітектура програми
Программа реалізована з використанням архітектурного підходу розділення відповідальностей:

Program - головний клас, відповідає за інтерфейс користувача

PlantCollection - клас для роботи з колекцією рослин

Plant - клас, що представляє рослину

PlantType - перелік типів рослин

Основні функціональності
Додавання рослин (вручну, випадковим конструктором, через парсинг)

Перегляд всіх рослин

Пошук рослин за різними критеріями

Видалення рослин

Демонстрація поведінки рослин

Робота зі статичними методами

Використання колекції List<T>
У програмі використовуються наступні методи List<T>:

Add() - для додавання елементів

RemoveAt() - для видалення за індексом

RemoveAll() - для видалення за умовою

Where() - для пошуку елементів

ToList() - для перетворення результатів

Clear() - для очищення колекції

Тестування
Створено два тестових класи:

ProgramTests - тестує методи роботи з колекцією

PlantTests - тестує функціональність класу Plant
